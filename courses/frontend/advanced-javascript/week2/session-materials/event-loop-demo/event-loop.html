Event loop

Code
Call stack ? Where synchronous code runs, one execution at a time. When the stack is empty, the event loop can move a callback from the async queue here.
Async queue ? Callbacks from setTimeout, I/O, events, etc. wait here. When the call stack is empty, the event loop moves one callback from this queue to the call stack.
Timer (Web APIs) ? When you call setTimeout(fn, ms), the callback fn is registered here. After ms milliseconds it moves to the async queue. Part of the browser's Web APIs.
